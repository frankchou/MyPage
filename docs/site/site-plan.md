## 二、雙語境切換（核心機制）

同一個頁面框架，兩種「語境」。切換時改變：背景影片 + 色溫 + 文案內容。
導覽列、版面、字型、動畫機制全部不變。

**切換觸發點**：原本 Prisma 的「Join the lab」按鈕位置，改為語境切換按鈕。

| 項目 | 語境 A：履歷（B2B） | 語境 B：創業（B2C + B2B 諮詢） |
|------|---------------------|-------------------------------|
| 按鈕文字 | → 探索我的創業 | → 看看我的經歷 |
| 背景影片 | Prisma 原始影片（暫用） | 業主另外提供（未提供前沿用同一支） |
| 色溫感覺 | 下午 / 黃昏 / 沉穩專業 | 早晨 / 晨光 / 活力探索 |
| 切換方式 | 影片 crossfade + CSS 變數切換色調 + 文案 fade out/in |
