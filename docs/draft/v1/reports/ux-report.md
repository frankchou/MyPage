## 核心設計決策摘要

1. **Drawer 優先於頁面跳轉**：所有節點展開均採頁內 Drawer，保護探索心流不中斷
2. **移除入場 Loading 動畫**：與 LCP < 1.5s 效能目標衝突，直接捨棄
3. **防複製功能完全移除**：違反使用者體驗原則，目標受眾高管更容易因此反感
4. **手機版需獨立設計**：雷達圖在 375px 無法呈現，改為文字挑戰清單功能等效替代
5. **技術圖加商業影響層**：架構圖需同時服務技術（CTO）與非技術（CEO）受眾

## 與 PRD 主要差異點

- 矩陣代碼 Hover 降階為邊框微亮
